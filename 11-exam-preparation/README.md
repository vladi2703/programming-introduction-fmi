# Задача 1 - Riddle

Въвежда се цяло число `N`, където `N` е размер на квадратнта матрица символи. Трябва да проверите дали въведен от входа низ (с дължина не повече от N) се съдържа в ред или колона на матрицата. 

Пример: 

```
5
a b c d e
f g h i
k l m n o
p q r s t
u v w x y
```
Низове:
rst 
oty 
ejoty
afkpu
hi
xy
